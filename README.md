# CityGML의 UML to XSD 변환을 위한 응용 프로그램

## 의존성(Dependency)
- Java Runtime Environment 11 or later

### CMD 명령어

```bash
java -jar ShapeChange-2.11.0.jar -Dfile.encoding=UTF-8 -c ShapeChangeConfigurationFile_NDTTransportation.xml
```
